#  Automation-Toolbox

A collection of high-quality, production-ready Python automation scripts. This repository serves as a technical demonstration of clean coding practices, efficient logic, and robust error handling.

##  Included Scripts

### 1. `site_sentinel.py`
**Website Health & Speed Check**
- Connects to multiple external APIs.
- Measures network latency in milliseconds.
- Implements robust error handling for connection timeouts.

### 2. `smart_scraper.py`
**Web Scraping & Data Extraction**
- Extracts live headlines using the `BeautifulSoup` library.
- Demonstrates HTML parsing and dynamic content filtering.

### 3. `table_tailor.py`
**Automated CSV Data Cleaning**
- Performs data normalization (Proper Case, Lowercase emails).
- Formats financial data into readable currency strings.
- Efficiently handles large-scale file I/O operations.

---

##  Usage
All scripts are designed to run with Python 3.
```bash
pip install requests beautifulsoup4
python [script_name].py
```
